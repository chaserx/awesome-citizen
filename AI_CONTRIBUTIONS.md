# AI-Assisted Contributions Guide

_Guidance for using AI tools when contributing to this awesome list_

AI assistants like GitHub Copilot, Claude, and ChatGPT can be helpful when contributing to this project. This guide explains how to use them effectively while maintaining the quality standards that make awesome lists valuable.

---

## Core Principles

- **Human Curation**: The value of an awesome list comes from thoughtful human selection. AI can help, but editorial judgment remains yours.
- **Verification Required**: Always verify that resources exist, work, and are relevant before submitting.
- **Transparency**: Be open about AI usage when it's significant to your contribution.

---

## Good Uses for AI

AI tools can help with:

- **Finding resources**: Discovering civic tech organizations, government APIs, or related projects
- **Formatting**: Converting links to proper markdown format, fixing title casing
- **Writing descriptions**: Drafting concise descriptions for resources you've found
- **Checking work**: Reviewing spelling, grammar, and markdown syntax
- **Research**: Understanding what a resource does to write an accurate description

---

## What AI Cannot Replace

- **Editorial judgment**: Deciding whether a resource belongs on this list
- **Quality evaluation**: Assessing if a resource is well-maintained and useful
- **Relevance assessment**: Determining if something fits the civic engagement focus
- **Link verification**: Confirming URLs work and point to the right content
- **Duplicate checking**: Verifying a resource isn't already listed

---

## Quality Checklist

Before submitting an AI-assisted contribution:

- [ ] Visited each link to confirm it works
- [ ] Verified the resource matches its description
- [ ] Checked the resource isn't already on the list
- [ ] Confirmed the resource fits the civic engagement theme
- [ ] Reviewed formatting matches existing entries
- [ ] Used AP-style title casing

---

## Disclosure

Mentioning AI assistance in your PR is optional but appreciated for substantial contributions. A simple note like "Used Claude to help draft descriptions" is sufficient.

---
